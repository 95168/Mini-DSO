# Mini-DSO
This is a simple oscilloscope made with STC MCU. 

You can use this Mini DSO to observe waveform.

Time Interval: 100us-500ms

Voltage Range: 0-30V

Draw Mode: Vector or Dots

![image](/pic/IMG_6270.JPG)

Material list:
![image](pic/IMG_6243.jpg)
